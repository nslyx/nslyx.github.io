http

发表于 2017-03-15

当浏览者访问一个网页时，浏览者的浏览器会向网页所在服务器发出请求。当浏览器接收并显示网页前，此网页所在的服务器会返回一个包含HTTP状态码的信息头（server header）用以响应浏览器的请求。

HTTP状态码的英文为HTTP Status Code。 下面是常见的HTTP状态码：

200 - 请求成功
301 - 资源（网页等）被永久转移到其它URL
404 - 请求的资源（网页等）不存在
500 - 内部服务器错误

PHP框架协作组(PHP-FIG)

发表于 2017-03-11

在PHP框架介绍中PSR规范时常被提及，PhpStorm IDE 也有对该规范的支持。So, just learn something about it.

PHP-FIG — PHP Framework Interop Group，直译为PHP框架互操作组，互操作可以理解为协作，所以就是PHP框架协作组。官网说它旨在通过合作和标准推进PHP的发展。其实就是对通过协商制定一些规范从而推进PHP框架的规范化进程，有助于提升PHP框架易用性与促进PHP框架的快速发展。PHP Standards Recommendations，即PSRs系列标准规范，也就是这个组织的成果了。
