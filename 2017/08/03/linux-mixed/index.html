Aug 3, 2017

Contents

  1. vim 编辑器

Linux 系统，话说我也不知道该怎么说这玩意，反正就是用着用着就不想它是个什么东西了，只要记住是个操作系统就行了，跟 windows 性质一样，是个计算机系统。Linux 属于 unix 派系，下面的分支众多，反正常用的(我听说过的)也就几种，因为一大堆乱七八糟的理由，这玩意普遍用作服务端系统，里面有很多道道是需要掌握和学习的，了解了这玩意儿，就会发现与之前 玩windows时 被灌输的诸多认知有多么大的差异。

vim 编辑器
