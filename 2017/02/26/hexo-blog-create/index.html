看过不少说什么教你用Github搭建免费博客的文章，其实就是利用那些Github的pages服务。
国内类似的还有Coding和开源中国的码云，都有pages服务，也都可以用于构建博客。需要注意的是这几家的pages服务不是说完全一致的，包括默认部署分支，项目部署地址，源的大小限制什么的可能都不大一致，详情参见文档Github Pages文档、Coding Pages文档和码云Pages文档。
