phper

Mar 7, 2017

我觉得很多从业者或者是招聘单位都像我一样，根本就没有一个客观可量化的标准来区分初中高级的程序员。

知乎答案原文：

基础：熟悉C，熟悉c++和java中至少一个，数据结构，算法有一定了解，英语过关。
初级：熟悉oop，了解常见的设计模式，能看懂别人的代码，世面上的cms和framework拿来即用。知道php不能做什么。
中级：能设计模块，熟悉队列缓存分布式。
高级：架构的设计，团队的管理。

其实我并不觉得这个答案足够具体，希望我能找到我自己的答案。
