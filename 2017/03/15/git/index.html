Git

版本控制器是在软件开发过程中帮助多人协作和版本控制的软件，目前最常见的版本控制器有 svn 和 git 。

目前比较好的第三方GitServer有Github，GitLab，Coding 和 码云。

下面说下对Git的概念理解。

  • 代码仓库
  • 分布式
  • 基于行的差异算法
  • 托管
  • 基本构成工作区、暂存取和本地仓库
  • 本地仓库和远端仓库
