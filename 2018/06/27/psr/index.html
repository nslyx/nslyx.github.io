PHP Framework Interop Group

PHP框架互操作组(PHP Framework Interop Group) 是PHP语言开发通用规范(PSR 规范)的采集和确立项目组织，旨在通过协作和标准推动PHP向前发展。任何 PHPer 都可以通过 GitHub 向该组织提出规范草案，如果提出的规范草案被广泛提到，就会在将来的 PSR 规范中采纳。大概就酱紫、

PSR

PSR(PHP Standards Recommendations) 是针对解决 PHP 编码混乱的情况而起草和明确的编码规范。在此之前，不同的 PHP 框架往往在目录结构、代码风格、常见机制实现等方面有较大的差异，学习任何任何一个框架都几乎需要从零开始，而 PSR 规范的广泛应用正在逐步改变这种现状。
